Typography & Emphasis

You can easily style text to add emphasis: * This text is bold (using double asterisks). * This text is italicized (using single asterisks). * This text has a ~~strikethrough~~ applied. * You can also combine them to make text bold and italic.

Code Snippets

Inline Code

To display a variable name or a short command within a sentence, wrap it in backticks like const user = "John";.

Fenced Code Block

For larger blocks of code, use three backticks before and after the block. You can also add the language name for syntax highlighting:

function greetUser(name) {
    console.log(`Hello, ${name}! Welcome to Markdown.`);
}

greetUser("Developer");
